# goRequest

An app to test your APIs using the SFML library.


### Install cmake
```shell
sudo apt-get install cmake
```
### Create the binary
```shell
cmake -S . -B build/
cd build/
make
cd ..
```
### Launch the project
```shell
./goRequest
```
